# FreeCAD Plugins and Macros

A curated collection of FreeCAD macros and plugins focused on **parametric modeling**, **woodworking**, and **cabinet design** workflows.

This repository is intended for:
- Woodworkers
- Furniture designers
- Makers
- Advanced FreeCAD users
